[128. 最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/)

给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

**示例 1：**

```
输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。
```

**示例 2：**

```
输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9
```

**示例 3：**

```
输入：nums = [1,0,1,2]
输出：3
```

 

**提示：**

- `0 <= nums.length <= 105`
- `-109 <= nums[i] <= 109`

这里我们采用思路是先排序再对比，对比思路是记录每段连续子序列的长度，然后更新得到最长的，如果数组nums[i]==nums[i+1] 直接跳过就行，满足 nums[i] ==nums[i+1]-1我们就对当前子序列长度+1，如果不满足则重新开始计数长度，每次我们可以对最大长度进行更新，注意默认长度应为1。

时间复杂度：`O(n log n)`

空间复杂度：`O(n)`（因为复制了一份数组）

```c++
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        int length = nums.size();
        if(!length) return 0;
        int ans = 1;
        int cur = 1;
        vector<int> sortNums = nums;
        sort(sortNums.begin(), sortNums.end());
        for(int i = 0; i < length - 1; i ++){
            if(sortNums[i] == sortNums[i + 1] - 1)
                cur ++;
            else if(sortNums[i] == sortNums[i + 1])
                continue;
            else
                cur = 1;
            ans = max(ans, cur);
        }
        return ans;
    }
};
```
